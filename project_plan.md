# Freelance Photographer Portfolio Website

## 1. Project Description
A personal portfolio website for a freelance photographer, videographer, and beginner web designer who is an active-duty military member. The site positions the creator as reliable, affordable, and professional. Target audience: individuals and small businesses seeking affordable content creation services.

## 2. Page Structure
- `/` - Home (single-page with anchor sections: Hero, Portfolio, About, Services, Trust, Contact)

## 3. Core Features
- [x] Hero section with CTA buttons
- [x] Portfolio grid with category filter (Photography, Video, Web Design)
- [x] About section with personal story and disclaimer
- [x] Services section (Photography, Videography, Web Design, Candid)
- [x] Trust/social proof section
- [x] Contact section with form and direct contact info
- [x] Responsive design (mobile + desktop)
- [ ] Contact form (functional via Readdy form URL)

## 4. Data Model Design
No database required — static portfolio site with form submission.

## 5. Backend / Third-party Integration Plan
- Supabase: Not needed
- Shopify: Not needed
- Stripe: Not needed
- Form: Readdy form endpoint for contact form submissions

## 6. Development Phase Plan

### Phase 1: Full Homepage Build
- Goal: Build the complete one-page portfolio site
- Deliverable: Hero, Portfolio, About, Services, Trust, Contact, Footer sections fully designed and functional

### Phase 2: Portfolio Enhancements (future)
- Goal: Add lightbox for portfolio images, video embed support
- Deliverable: Interactive gallery with video previews
